## Running Locally

This application requires Node.js v18.13+.

```bash
npm install -g @angular/cli
git clone https://github.com/kaiyoken3618/Portfolio
cd kevinflor.es
npm install
```

run:
```bash
 npm start
```

Wait to compile and go to http://localhost:4200 after compile finish
